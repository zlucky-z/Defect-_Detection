# PCB缺陷检测系统

基于深度学习的高性能PCB缺陷检测系统，集成YOLOv5批量检测和YOLOv8实时流检测，专为工业生产线设计。

## 🎯 核心特性

- **双模型架构**：YOLOv5批量离线检测 + YOLOv8实时RTSP流检测
- **硬件加速**：基于Sophon BM1684X芯片，实现22+ FPS实时检测，延迟仅30ms
- **Web统一管理**：现代化Web界面，支持用户权限、检测控制、历史记录、数据分析
- **流媒体集成**：内置MediaMTX服务器，支持RTSP流推送和多终端观看
- **智能优化**：跳帧处理、低延迟模式、自动流管理

## 📋 功能模块

| 模块 | 功能 |
|------|------|
| 批量检测 | 支持多图片上传，离线批量处理 |
| 实时监控 | RTSP视频流实时检测，可视化结果输出 |
| 历史管理 | 检测记录查询、结果回溯 |
| 数据分析 | 缺陷统计、趋势分析、可视化报表 |
| 系统监控 | 资源占用、性能指标实时监控 |

## 🔍 检测类别

- **连锡缺陷**（link）：焊接连锡问题
- **未知缺陷**（unknown）：其他异常情况

## ⚡ 性能指标

- 实时检测速度：≥22 FPS
- 单帧处理延迟：~30ms
- 视频分辨率：1920×1080
- 置信度阈值：可配置（默认0.3）
